[Express](http://expressjs.com/) 
[Passport](http://passportjs.org/) 

## Instructions



```bash
$ git clone https://github.com/odiliomob/express-passport.git
$ cd express-passport
$ npm install
```

Start the server.

```bash
$ node server.js
```

 [http://localhost:3000/](http://127.0.0.1:3000/)
