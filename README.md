[Express](http://expressjs.com/) 
[Passport](http://passportjs.org/) 

## Instructions



```bash
$ git clone git@github.com:passport/express-4.x-local-example.git
$ cd express-4.x-local-example
$ npm install
```

Start the server.

```bash
$ node server.js
```

 [http://localhost:3000/](http://127.0.0.1:3000/)
